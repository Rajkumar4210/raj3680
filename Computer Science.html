Input means data and instruction given to the computer which is most essential for producing meaningful and useful output.The unit which is used to give input to the computer system is called input unit and it is formed by the various input devices attached to the computer such as keyboard ,mouse,joystick,trackball,light pen, touch screen,MICR (Magnetic Ink Character Reader),OMR (Optical Mark Reader) etc.


Input unit carried out following functions :
  • It accepts/reads instrutions and data from outside world.
  • It suppies the coverts instruction and data to the storage unit for storage and futher processing.
  • It converts these instructions and data in computer acceptable form. Unit called input interfaces accomplish this task.
